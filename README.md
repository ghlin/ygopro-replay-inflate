# Replay Inflater

expands a `.yrp` replay file(generated by ygopro) into json file contains the commands of each step.

e.g.:

``` json
{
  "meta": {
    "players": [
      {
        "type": "host",
        "name": "yugi",
        "deck": {
          "main": [ ... ],
          "extra": [ ... ]
        }
      },
      {
        "type": "client",
        "name": "kaipa",
        "deck": {
          "main": [ ... ],
          "extra": [ ... ]
        }
      }
    ],
    "settings": {
      "start_lp": 4000,
      "start_hand": 6,
      "rule": "master-rule-4",
      "shuffle_deck": true,
      "draw_count": 1
    }
  },
  "steps": [
    {
      "command": "duel-start"
    },
    {
      "command": "initial-hand",
      "player": 0,
      "cards": [ ... ]
    },
    {
      "command": "initial-hand",
      "player": 1,
      "cards": [ ... ]
    },
    {
      "command": "turn-change",
      "player": 0,
      "turn": 1
    },
    {
      "command": "phase-change",
      "player": 0,
      "phase": 0
    },
    {
      "command": "phase-change",
      "player": 0,
      "phase": 1
    },
    {
      "command": "draw-card",
      "player": 0,
      "card": ...,
      "public": "false"
    }
    , ...
  ]
}
```
